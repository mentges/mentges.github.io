# Mentges AI Enablement

This repository contains the source code for the landing page of **Mentges AI Enablement**, an enterprise solutions company specializing in highly-critical industrial sectors such as Oil & Gas and Civil Engineering in Brazil.

**Website:** [mentges.github.io](https://mentges.github.io)

## Project Structure

The project was recently refactored from a legacy Jekyll framework into a modern, pure static architecture (HTML5/CSS3/Vanilla JS) to guarantee maximum performance, direct stylistic control, and top-tier SEO. The styling relies on a premium dark-mode approach utilizing *Glassmorphism* layered effects.

- **`index.html`**: The main structural and semantic markup of the landing page.
- **`css/style.css`**: Complete styling suite, including CSS variables, glass-panel visual effects, custom grid layouts, and media queries.
- **`js/main.js`**: Core logic for the mobile navigation toggle and the Intersection Observer API driving smooth scroll-based animations.
- **`assets/img/`**: Stores static image assets, such as the Bacalhau FPSO platform impact rendering.
