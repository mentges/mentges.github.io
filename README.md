# Bruno Mentges / Mentges.AI

Personal consulting website for **Bruno Mentges** — Senior Data & AI Engineer with 28 years of production experience. Fractional architecture, AI integration, and data platform design for international teams.

**Live site:** [mentgesinformatica.com.br](https://mentgesinformatica.com.br) / [mentges.github.io](https://mentges.github.io)

## What this is

A static consulting website with a technical blog. No frameworks, no build tools — just HTML, CSS, and vanilla JS hosted on GitHub Pages.

The site serves two purposes:
1. **Consulting presence** — services, track record, case studies, booking link
2. **Content engine** — 8 technical articles with 35+ hand-crafted SVG diagrams

## Design: "Ink & Signal"

A refined editorial theme with technical undertones:

- **Typography**: Instrument Serif (headings), Switzer (body), JetBrains Mono (labels/tags)
- **Palette**: Deep ink navy, copper/amber accent, teal secondary, cream backgrounds
- **Visual language**: Custom SVG diagrams throughout — data flow visualizations, architecture diagrams, decision trees, comparison charts. All inline, no external image dependencies.
- **Texture**: Dot grid patterns, noise overlay, CSS grid patterns in dark sections

## Structure

```
/                              Homepage — hero, about, services, track record, blog preview, CTA
/blog/                         Blog index — 8 published articles
/blog/[article].html           Individual blog posts with inline SVG diagrams
/case-studies/[study].html     3 anonymized case studies
/css/style.css                 Main theme (Ink & Signal)
/css/pages.css                 Inner page styles (articles, case studies)
/js/main.js                    Mobile menu, scroll animations, navbar behavior
```
